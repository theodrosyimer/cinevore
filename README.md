# Cinevore

## Context

Formation Assofac DWWM 2022-2023
> Projet de réseau social autour du cinéma.

## Usage

**Install**

```sh
git clone https://github.com/theodrosyimer/groupomania
```

**Start**

To start the client and server:

```sh
npm start
```
