# Cinevore

## Context

Formation Assofac DWWM 2022-2023
> Projet de réseau social autour du cinéma.

## Usage

**Install**

```sh
git clone https://github.com/theodrosyimer/cinevore
cd cinevore
npm install
```

**Development**

To start the app in development mode:

```sh
npm run dev
```

**Start**

To start the app:

```sh
npm run build && npm start
```
